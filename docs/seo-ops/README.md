# SEO/content operations

Thư mục này lưu report và bản nháp nội bộ do script hoặc Codex Schedule tạo ra.

- `reports/`: audit SEO, internal link, content gap và refresh candidates.
- `drafts/`: bản nháp tiếng Việt cần duyệt thủ công trước khi chuyển thành nội dung publish.

Nguyên tắc:

- Không tự động publish.
- Không tự động deploy.
- Không dùng paid API.
- Không dùng public game/comics API để giả làm dữ liệu slot thật.
- Không bịa provider, RTP, jackpot, payout, giấy phép hoặc khuyến mãi.
- Nội dung iGaming phải có cảnh báo 18+ và chơi có trách nhiệm.
