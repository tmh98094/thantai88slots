# Thantai88Slots

Standalone Vietnamese slot/iGaming project for `thantai888.co`.

- Next.js app router
- Mobile-first red/gold casino design
- GSAP entrance animation layer
- Sticky affiliate CTA using `/go/platform`
- Public site content in Vietnamese
- Affiliate target defaults to `https://www.thantai688.com/?f=55`

Run locally:

```powershell
npm install
$env:NEXT_PUBLIC_SITE_URL='https://thantai888.co'
npm run build
npm run start -- -p 3002
```
